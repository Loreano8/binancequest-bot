## The Problem

Millions of people join Binance every year without truly understanding crypto. Existing educational content is scattered, dry, and hard to stick with. There's no system that rewards learning or builds daily habits.

## The Solution

BinanceQuest AI turns crypto education into an engaging quest:

- 🎯 **Daily quizzes** based on Binance Academy content
- ⚡ **XP system** — earn points, level up, unlock badges
- 💰 **Live prices** from the official Binance API
- 🤖 **AI assistant** that answers any crypto question
- 📰 **Market briefings** generated by OpenClaw AI
- 💼 **Portfolio tracker** with AI rebalancing suggestions
- 🌍 **Bilingual** — French 🇫🇷 and English 🇬🇧

## How It Works

```
User sends /quiz
       ↓
OpenClaw processes request
       ↓
BinanceQuest skill selects random question from Binance Academy bank
       ↓
User answers → Gets XP + explanation
       ↓
Daily habit formed 🎯
```

